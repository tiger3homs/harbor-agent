# Contributing to structured-clause-agent

Thank you for your interest in contributing!

## Development Setup

1. Clone the repo
2. `npm install`
3. `npm run build`
4. `npm run dev`

## Code Style

- TypeScript strict mode
- Prefer functional style where possible
- Keep functions small and focused
- Add JSDoc for public APIs

## Pull Request Process

1. Create a feature branch from `main`
2. Make focused, atomic commits
3. Update `README.md` and `CHANGELOG.md` when appropriate
4. Ensure `npm run build` succeeds
5. Open a PR with a clear description

## Areas Where Help Is Needed

- Full Claude Code hook integration
- Vector search (Chroma) support
- TUI / interactive CLI
- Real AST-based CodeCompressor
- Testing suite
- Documentation & examples

## Questions?

Open an issue or start a discussion.
